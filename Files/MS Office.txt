commands to execute programs:

Python: 
python filename.py

C: 
gcc filename.c -o executeablefilename.exe --> Compilation
executeablefilename.exe --> Execution

C#:
csc filename.cs --> Compilation
filename --> Execution

VisualBasic:
vbc filename.vb --> Compilation
filename --> Execution

Go:
go run filename.go

Java:
javac filename.java --> Compilation
java classname --> Execution

NodeJS:
node filename

NestJS:
npm start
(or)
npm run start:dev
