commands to execute programs:

Python: 
python filename.py

Python(Linux):
sudo apt install python
python3 filename.py
pip3 modulename

C: 
gcc filename.c -o executeablefilename.exe --> Compilation
executeablefilename.exe --> Execution

C#:
csc filename.cs --> Compilation
filename --> Execution

VisualBasic:
vbc filename.vb --> Compilation
filename --> Execution

Go:
go run filename.go

Java:
javac filename.java --> Compilation
java classname --> Execution

NodeJS:
Create commands:
npm i express-generator
npx express --view=ejs sample
Run commands:
node filename
npx nodemon filename

NestJS:
Create commands:
sudo npm i -g @nestjs/cli
nest new project-name
Run commands:
npm start
npm run start:dev

nest g co name
nest g s name
nest g mo name

ReactJS:
npm install -g create-react-app
npx create-react-app projectname
